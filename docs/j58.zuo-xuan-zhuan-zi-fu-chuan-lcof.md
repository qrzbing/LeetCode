# 剑指 Offer 58 - II. 左旋转字符串

Difficulty: Easy
Language: C++
Link: https://leetcode-cn.com/problems/zuo-xuan-zhuan-zi-fu-chuan-lcof/
No: 58
Status: Done
Tags: 字符串

-----
```cpp
class Solution
{
public:
    string reverseLeftWords(string s, int n)
    {
        string ret_str;
        for (int i = n; i < s.size(); ++i)
        {
            ret_str += s[i];
        }
        for (int i = 0; i < n; ++i)
        {
            ret_str += s[i];
        }
        return ret_str;
    }
};
```

这道题也用了取巧的办法，没有修改原字符串而是创建了一个新字符串
